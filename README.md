# CBSS @ NYU SPS Website

This is the static website for the Consulting & Business Strategy Society (CBSS) at NYU SPS.

## Current status

The site has been updated with a first live content pass using public CBSS information, including:

- organization overview
- official contact email
- public links
- current and recent events
- public-facing connection channels

## Project structure

```text
.
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to run locally

1. Clone or download the project.
2. Open `index.html` in your browser.

No build tools or dependencies are required.
